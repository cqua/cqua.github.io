hi all,
ive been trying a new style of 3d modeling the style is inspired by kaykit, a popular free asset library.

smooth shading. matte texture where my previous style required using different textures for different parts of the mesh, this new style uses one texture for every mesh. the texture is a palette of gradients. rather than drawing details like eyes in a texture, those details are all modeled. this gives the mesh a clay-like appearance. using the gradients, you can also create the illusion of shading which looks nice in any lighting. you can play the gradient towards the camera. gradient also looks like baked clay 1. clay
