Welcome to the Doors of Illune Campaign Guide

Characters will start at level 0. To start, pick an ancestry and follow its instructions. If you can, take a look at the combat overview. SotDL is similar to D&D in a lot of ways, but the combat has some unique quirks.

Refer to the Advancement table for what to do upon leveling up:

LevelBenefit
0Choose an Ancestry
1Choose a Novice Path
2Novice Path enhancement
3Choose an Expert Path
4Ancestry enhancement
5Novice path enhancement
6Expert path enhancement
7Choose a Master Path OR a second Expert Path
8Novice path enhancement
9Expert path enhancement
10Master path enhancement
